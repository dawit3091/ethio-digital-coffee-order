# ethio-coffee
Origins Buna 
A sleek, responsive frontend landing page for a modern coffee shop. This project focuses on clean design, custom asset integration, and a seamless user experience across all devices.
Features
Responsive Design: Fully optimized for mobile, tablet, and desktop screens.

Modern UI/UX: Clean aesthetics with custom-curated visuals that match a warm, premium coffee shop vibe.

Interactive Elements: Smooth navigation, dynamic hover effects, and a structured menu layout.

Tech Stack
HTML5: Semantic markup for structure and SEO.

CSS3: Custom styling, Flexbox/Grid layouts, and media queries for responsiveness.

JavaScript: Minimal, clean script for interactive UI elements (e.g., mobile menu toggle).
